3.更新记录

2022/02/13 版本 3.0.0

  • 现在，配置文件中的字体大小以 pt 为单位。
    因此，第二版和第三版程序的字体配置文件不可互相替代。
  • 现在，语言资源文件用 UTF-8 编码。
  • 现在，设置按钮的文字更易于理解。

2021/11/28 版本 2.42(目前只在 GitHub 上发布)

  • 添加了 Windows 11 的系统默认字体。
  • 现在，Windows 11 的版本号会正常显示了。

2021/01/10 版本 2.41

  • 现在能够读取各字体下所有的字形了。
  • 支持显示Windows 10 Version 20H2的版本号。

2020/07/26 版本 2.40.2

  • 改进韩语翻译。

2020/06/28 版本 2.40.1

  • 改进韩语翻译。

2020/05/31 版本 2.40

  • 新增了应用按钮。当点击该按钮后将应用设定且不关闭程序。
  • 将主界面分成全局设定与单项设定两组。
  • 修复了无法读取当前目录内配置文件的错误。

2020/01/13 版本 2.36

  • 当点击统一设定后将自动关闭程序。
  • 添加葡萄牙语（巴西）翻译。
  • 添加繁体中文翻译。
  • 包括2.35版本中忘记加入的简体中文翻译。

2018/04/28 版本 2.35

  • 当Windows系统显示语言与区域设置不同时，以显示语言来决定软件语言。
  • 增大软件各处的字体大小。
  • 添加了韩语的界面翻译。感谢deVbug的协助。
  • 支持韩语字体名替换（仅限韩语。）
  • 改进了简体中文的翻译品质。感谢kingofotaku的协助

2017/06/03 版本 2.34

  • 启动后主界面将会在屏幕中心出现。
  • 字体选择对话框将出现在主界面的上部。
  • 帮助文档将会在主界面的左上方打开。
  • 修复简体中文帮助文档内字体显示异常。

2017/05/14 版本 2.33

  • 在Windows 10系统上运行时可显示详细系统版本信息。
  • 恢复系统默认字体功能支持英文版。
  • 在语言文件中添加默认字体及字符集。
  • 新增根据系统区域设置选择相对应的语言及帮助文档。今后新增翻译时将无需修改程序，并可单独发布语言文件。

2016/10/10 版本 2.32

  • 添加了简体中文版帮助文档。感谢立花 月下的协助。
  • 恢复系统默认字体功能支持简体中文版。
  • 变更源文件中文件夹的结构，以便未来添加针对更多语言的翻译。

2016/09/19 版本 2.31

  • 添加了简体中文的界面翻译。感谢立花 月下的协助。
  • 帮助文档已变更为HTML格式。
  • 在版本信息中添加了翻译者名单。

2016/08/19 版本 2.30

  • 多语言预览版。

更早的版本更新记录请参阅日文版帮助文档。
