1.快速入門

1.1 這是什麼？

這個軟體可以設定 Windows 8.1/10 上的系統字型。

1.2 目標作業系統

這個軟體可以執行於 Windows 8、8.1 和 10。

1.4 安裝

安裝

將所有檔案解壓縮到一個資料夾裡。

卸載

刪除所有檔案。

1.5 Trouble shooting

Before change system fonts

If you want to change system fonts by this program, I recommended to backup registry by utils\SAVEreg.bat (required Administrator privilege).
Backup file is created on C:\Users\(user name)\AppData\Local.

If trouble occurred

If trouble occurred and you took backup by utils\SAVEreg.bat, run utils\RESTOREreg.bat (required Administrator privilege).
You can restore registry data before changing system fonts.
